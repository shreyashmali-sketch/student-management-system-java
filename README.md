# Student Management System (Java Console)

A simple menu-driven Java console application built using Object-Oriented Programming (OOP) concepts.

## Features
- Add student
- View all students
- Delete student by ID

## Concepts Used
- Java OOP (Class, Object, Encapsulation)
- ArrayList
- Scanner for input
- Menu-driven program

## Project Structure
- Main.java → Handles menu and user input
- Student.java → Represents student entity
- StudentManager.java → Manages student operations

## How to Run
1. Open the project in IntelliJ IDEA
2. Run `Main.java`
3. Follow on-screen menu options
